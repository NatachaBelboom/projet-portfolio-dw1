## Look and feel

Les couleurs du site resteront simples. C'est-à-dire pour la base du noir, voire du gris foncé et du blanc avec quelques touches de couleurs.
Les titres sont prévus en grandes tailles 
Tout les boutons se colorent au survol
 
## Stratégie
1. Page d'accueil

L'utilisateur arrive direct sur le dernier projet réalisé. Cela donne envie de le découvrir avant de naviguer plus loin dans le site.
Même stratégie utilisée avec la phrase d'accroche à propos de moi.

* Section projets

Les projets sont disposés un peu comme une galerie photo. Les photos présentant les projets sont rectangulaires avec le titre correspondant dans le bas de l'image.
Le fond est plutot sobre pour faire ressortir les projets réalisés.

* Section footer

Les liens vers mes réseaux sociaux sont colorés. 
La navigation est aussi présente
    
2. Page d'un projet

Avec cette page, on en apprend un peu plus sur un projet spécifique grâce aux images du site et une brève explication. 
    
3. Page à propos

Biographie reprenant mon parcours, mes compétences. 
Une photo et des liens sont présents. Un vers mes projets et un autre vers la page de contact
  
4. Page de contact

Formulaire de contact avec les champs: adresse mail, sujet, message
